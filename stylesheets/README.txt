ВНИМАНИЕ: проект написан с помощью препроцессора SASS и библиотеки COMPASS 
НЕ РЕКОМЕНДУЕТСЯ вносить правки в файлы с расширениями .css
Писать код для стилей только в файлах с расширениями .scss (эти файлы находятся в папке /sass)
Компиляцию scss в css проводить следующим образом:

1. Если на сервере есть Ruby достаточно прописать в консоли установку следующих гемов:
	gem install sass 
	gem install compass

   и затем компилировать файлы при помощи команды:
	compass watch

Это все, - теперь можете писать сss-правила в файл c расширением .scss
Компиляция будет происходить в автоматическом режиме.

2. Если Ruby нет, - нужно установить Ruby на компьютер, затем воспользоваться утилитой expandrive или ей подобной, слить файлы проекта с сервера.
После выполнить условие 1.

3. Если для вас вышеперечисленное представляет кардинальные сложности, - вы можете править файлы .css
Но помните: при первой же компиляции файлов scss в css ВСЕ подобного рода ПРАВКИ будут СТЕРТЫ. Я вас предупредил.
